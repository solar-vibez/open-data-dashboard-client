# open-data-dashboard-client
Interactive dashboard for open data visualization [Front-end]

## Tech Stack
### React + TypeScript + Vite + SWC
This project is built on template that provides a minimal setup to get React working in Vite with HMR.
The build is done with SWC + plugin:
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

### Tailwind + Shadcn/ui

[Shadcn/ui](https://ui.shadcn.com/) is used as UI library.
